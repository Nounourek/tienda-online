Tienda Online - Proyecto en Angular

Este proyecto es una tienda en línea sencilla construida con Angular.

Características:
- Los usuarios pueden authenticarse, registrarse, acceder a la parte cliente y la parte admin de la tienda.
Instrucciones para ejecutar el proyecto:
1. Clona el repositorio:
   git clone https://github.com/Nounourek/tienda-online.git
2. Entra en el directorio del proyecto:
   cd tienda-online
3. Instala las dependencias:
   npm install
4. Corre la aplicación:
   ng serve
